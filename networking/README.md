# Enterprise Network Diagnostics & Security Checker

Enterprise Network Diagnostics & Security Checker is a Linux-focused toolkit designed to validate network health, security, and performance across enterprise environments.

## Key Features

- Layer-by-layer connectivity analysis from link to application
- Firewall rule audit with compliance-focused reporting
- DNS resolution validation and name-server verification
- Network performance baseline measurements
- Security port scan detection and exposure assessment

## Purpose

Built for operations and security teams, this tool helps identify issues quickly and delivers actionable insights for remediation and continuous improvement.
